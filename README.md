# GrowTechStack (GTS)

**GrowTechStack**은 흩어져 있는 국내외 기술 블로그의 양질의 콘텐츠를 한곳에 모아 제공하는 **기술 블로그 어그리게이터(Aggregator)** 서비스입니다. 

주요 기술 블로그(Toss, Kakao, Woowahan 등)의 RSS를 자동으로 수집하여 개발자들에게 최신 트렌드를 편리하게 전달하며, 향후 사용자가 직접 지식을 공유하고 성장할 수 있는 플랫폼을 지향합니다.

---


## Tech Stack

- **Backend**: Java 17, Spring Boot 3.x, Spring Data JPA, Spring Security
- **Database**: H2 (In-memory/File) (추후 MySQL로 마이그레이션 예정)
- **Frontend**: Thymeleaf, Bootstrap 5, JavaScript (추후 분리 예정)
- **Library**: Rome (RSS Parsing), SpringDoc (Swagger/OpenAPI)
